========================
`dorado` In Publications
========================

`dorado` has been applied in a number of peer-reviewed scientific publications.
We provide references to those publications below for users interested in learning more about the application of `dorado` in various scientific contexts.
If you have used `dorado` in your research, please let us know by opening an issue on the `dorado` GitHub repository and we'd love to add your publication to this list!

.. Note to contributors, we use a modified version of the APA citation style for consistency across publications, adding the DOI link to the end of the APA citation.
